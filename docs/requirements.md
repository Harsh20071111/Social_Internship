# Requirements Document

This document outlines the core requirements for the B2B Manufacturing Company Website.

## Overview
The website serves as a digital storefront for a manufacturing company with 3 separate factories, producing items like spoons, kitchenware, steel utensils, and household products.

## Key Features
- **Public Website:** Hero section, product catalog, factory showcase, about us, contact page.
- **Inquiry System:** Bulk inquiry form with WhatsApp and Email integration.
- **Admin Dashboard:** Manage products, categories, factories, and inquiries.
- **Tech Stack:** Next.js 16 (or latest), TypeScript, Tailwind CSS, shadcn/ui, Framer Motion, Prisma, PostgreSQL, Cloudinary, React Hook Form, Zod, Zustand, TanStack Query, Resend.

## Future Scalability
- Online ordering
- Dealer portal
- Distributor management
- Inventory tracking
- Analytics dashboard
- AI chatbot and product recommendation assistant
