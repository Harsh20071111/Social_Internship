# API Documentation

This document will outline the internal API routes for the B2B Manufacturing Company Website.

*To be populated as features are implemented.*

## Endpoints (Planned)
- `GET /api/products` - Fetch all products
- `POST /api/products` - Create a product (Admin)
- `GET /api/factories` - Fetch all factories
- `POST /api/inquiries` - Submit a new bulk inquiry
